# IMPORTANT: Before Running the Server

1. Create a `.env` file in this directory with the following content:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_mysql_password
DB_NAME=task_manager
DB_PORT=3306
PORT=5000
FRONTEND_URL=http://localhost:3000
```

2. Replace `your_mysql_password` with your actual MySQL password

3. Then run: `npm install` (if you haven't already)

4. Then run: `npm start`
